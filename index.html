Dataset 1: Azure Computer Vision for Image Classification

Dataset Information

  • This dataset consists of 6 types of cat breed:
    • American Shorthair
    • Bengal
    • Maine Soon
    • Ragdoll
    • Scottish Fold
    • Sphinx
  • Train data have 200 images for each class
  • Test data have 50 images for each class

Dataset 1: Azure Computer Vision for Image Classification

Data Source

  • The data gathered from Google Images, Shutterstock, and another image provider website with Google Chrome plug-in

Dataset 2: Azure ML for Tabular Data Classification

Dataset Information

  • This dataset contains ~1000 items with data on 3 different cat breeds:
    • Maine Coon
    • Ragdoll
    • Angora

Dataset 2: Azure ML for Tabular Data Classification

Additional Information

  • It includes information about animal's breed, age, gender, body length, weight, fur colour and pattern, eye colour, sleeping and playing time, country (including latitude and longitude), etc.
